# WebSocket

在应用层第七层上的一个应用层协议，实现服务器与客户端双向平等对话

它可以在用户的浏览器和服务器之间打开交互式通信会话。

向服务器发送消息并接收事件驱动的响应，而无需通过轮询服务器的方式以获得响应。

客户端会首先向服务器发送一个 HTTP 请求，包含一个 Upgrade 请求头来告知服务器客户端想要建立一个 WebSocket 连接

let websocket = new WebSocket('ws://127.0.0.1:3000')
//建立连接
websocket.onopen = function (event) {
  console.log('WebSocket is open now.')
}
//收到信息
//可以使用binaryType属性，显式指定收到的二进制数据类型
//ws.binaryType = "blob";
websocket.onmessage = function (event) {
  if (typeof event.data === String) {
    console.log('Received data string')
  }
  if (event.data instanceof ArrayBuffer) {
    var buffer = event.data
    console.log('Received arraybuffer')
  }
}
//关闭连接
websocket.onclose = function (event) {
  var code = event.code
  var reason = event.reason
  var wasClean = event.wasClean
  // handle close event
}
//发送数据
websocket.send('your message')
//发生错误时
websocket.onerror = function (event) {
  // handle error event
}

# Document.createDocumentFragment

说明：创建一个新的空白的文档片段

<div id="box">
      <ul id="ul"></ul>
    </div>
    <script>
      var element  = document.getElementById('ul'); // assuming ul exists
      var fragment = document.createDocumentFragment();
      var browsers = ['Firefox', 'Chrome', 'Opera',
          'Safari', 'Internet Explorer'];

      browsers.forEach(function(browser) {
          var li = document.createElement('li');
          li.textContent = browser;
          fragment.appendChild(li);
      });

      element.appendChild(fragment);
    </script>

# MutationObserver

说明：提供了监视对 DOM 树所做更改的能力

<b>
<div id="container">
      <div id="target">

      </div>
    </div>
    <button>变红</button>
    <button>变黑</button>
    <button>停止观察</button>
  </body>
  <script>
    const son = document.querySelector('#target')
    const btn = document.querySelectorAll("button")[0]
    const btn2 = document.querySelectorAll("button")[1]
    const btn3 = document.querySelectorAll("button")[2]
    btn.addEventListener('click', function() {
      son.style.backgroundColor = 'red'
    })
    btn2.addEventListener('click', function() {
      son.style.backgroundColor = 'black'
    })

    var observerOptions = {
      // childList: true,  // 观察目标子节点的变化，是否有添加或者删除
      attributes: true, // 观察属性变动
      // subtree: true     // 观察后代节点，默认为 false
    }
    //一个是描述所有被触发改动的 MutationRecord 对象数组，另一个是调用该函数的 MutationObserver 对象。
    let observe = new MutationObserver((mutationList, observer)=>{
      console.log(mutationList)
    })
    observe.observe(son,observerOptions)
    btn3.addEventListener('click', function() {
      observe.disconnect()
    })
  </script>

# Blob

说明：表示一个不可变、原始数据的类文件对象。它的数据可以按文本或二进制的格式进行读取，也可以转换成 ReadableStream 来用于数据操作。

# 属性

# size

Blob 对象中所包含数据的大小（字节） 1MB = 1024*1024 Byte

# type

Blob 对象包含数据的 MIME 类型，默认为‘’

# 方法

# slice 切片

创建一个包含源 Blob (opens new window)的指定字节范围内的数据的新 Blob (opens new window) 对象。

# stream

返回一个能读取 blob 内容的 ReadableStream

# text

返回一个 Promise 对象且包含 blob 所有内容的 UTF-8 格式的 USVString

# arrayBuffer

返回一个 Promise 对象且包含 blob 所有内容的二进制格式的 ArrayBuffer

# AbortController

表示一个控制器对象，允许你根据需要中止一个或多个 Web 请求。

# 属性

# signal

通过AbortController.signal (en-US) (opens new window) 属性获取到它的关联对象AbortSignal (opens new window) 的引用。

# AbortSignal 对象
# 属性
# aborted

表示与之通信的请求是否被终止（true）或未终止（false）。

# onabort

当 abort 事件触发时，即当信号正在与之通信的 DOM 请求被中止时调用。

# 方法

# abort

终止获取请求

const controller = new AbortController()
let signal = controller.signal

const downloadBtn = document.querySelector('.download')
const abortBtn = document.querySelector('.abort')

downloadBtn.addEventListener('click', fetchVideo)

abortBtn.addEventListener('click', function () {
  controller.abort()
  console.log('Download aborted')
})
